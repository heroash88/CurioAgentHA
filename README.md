# CurioAgentHA

Home Assistant add-on repository for [Curio](https://github.com/heroash88/curio-robot) -- a voice-powered AI assistant.

## Installation

1. Open Home Assistant
2. Go to **Settings** > **Add-ons** > **Add-on Store**
3. Click the three-dot menu (top right) > **Repositories**
4. Paste: `https://github.com/heroash88/CurioAgentHA`
5. Click **Add**, then find and install **Home Assistant App**

See [curio/README.md](curio/README.md) for full documentation.
